QueueSummary

Synopsis

Show queue summary.

Description

Request the manager to send a QueueSummary event.

Syntax

Action: QueueSummary
ActionID: <value>
Queue: <value>
Arguments
  • ActionID - ActionID for this transaction. Will be returned.
  • Queue - Queue for which the summary is requested.
