  • ${REDIRECTING_CALLEE_SEND_MACRO}
    Macro to call before sending a redirecting update to the callee
  • ${REDIRECTING_CALLEE_SEND_MACRO_ARGS}
    Arguments to pass to ${REDIRECTING_CALLEE_SEND_MACRO}
