Ping

Synopsis

Keepalive command.

Description

A 'Ping' action will ellicit a 'Pong' response. Used to keep the manager connection open.

Syntax

Action: Ping
ActionID: <value>
Arguments
  • ActionID - ActionID for this transaction. Will be returned.
